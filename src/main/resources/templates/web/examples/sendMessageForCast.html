Send Message example for Cast

How to use

  • This example shows how to send messages in a video broadcast but can be applied for video calls as well.
  • Enter a channel name and click the "CREATE" button to create a channel.
  • Open this link in another tab or a browser. Then click on the channel you created from the "Channel List".
  • When the broadcast begins, you can send a message by entering your username and your message.
  • You can read all the sent and received messages in the "Chat Log".
  • Source code: github page
  • Please use a unique channel name.
