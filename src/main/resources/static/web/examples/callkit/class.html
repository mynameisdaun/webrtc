How to use

  • 안드로이드 Remon Call Kit에 로그인을 합니다.
  • 로그인이 됐다면 왼족의 학생 리스트에 뜨게 됩니다.

  • 로그인시 서버에 푸시주소가 발행되며, DB에서 관리합니다.
  • 학생을 선택해서 누르게 되면, 해당 학생에게 전화를 걸게 됩니다.
  • 해당 학생의 앱에서 전화를 수락하면 통화가 시작됩니다.
  • 통화의 Channle ID 는 랜덤하게 생성됩니다.
